localStorage

Armazenamento local no browser.

Até 5MB. Formato de dados 'String'

'chave': valor => key: value

CRUD

  • [C] Create = criar -> .setItem(key, value)
  • [R] Read = ler -> .getItem(key)
  • [U] Update = atualizar -> .setItem(key, value)
  • [D] Delete = apagar -> .removeItem(key)

métodos do localStorage

  • .setItem(key, value)
  • .getItem(key)
  • .removeItem(key)
  • .clear()
